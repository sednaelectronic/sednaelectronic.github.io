Loop V1.0

Vehicle Detection System

← Back to Projects

Product Overview

Loop V1.0 is a distributed vehicle detection system that uses LiDAR distance measurement sensors to detect vehicle presence and wirelessly transmit detection data to a central master unit. The system provides vehicle counting capability per lane for traffic monitoring and control applications, making it ideal for compact deployment in short- to mid-range traffic monitoring scenarios. Designed for traffic control applications, the system provides reliable vehicle detection for traffic lights, traffic monitoring, and access control systems.

Technical Specifications

The Loop V1.0 system features a detection range of up to 10 meters, providing effective vehicle detection for typical intersection and lane monitoring applications. Network communication is based on NRF protocol operating at 2.4 GHz frequency, ensuring reliable wireless data transmission in urban environments. The system operates with sensor units requiring 12V at 1A current consumption, while the receiver unit operates at 12V with 2.5A current consumption.

Installation and Deployment: The maximum installation distance between each sensor and the receiver is 300 meters, enabling flexible sensor placement while maintaining reliable communication with the central unit. This range makes the system well-suited for compact intersection deployments and mid-range traffic monitoring applications where sensors can be strategically positioned within the communication range.

Purpose and Use Cases

The system enables real-time vehicle presence detection, traffic density estimation, traffic control logic input, intersection management, and access control based on vehicle detection. Its compact deployment characteristics and 10-meter detection range make it particularly suitable for short- to mid-range traffic monitoring applications where precise per-lane vehicle counting is essential.

Key Applications

  • Traffic light control at intersections
  • Vehicle counting and traffic flow analysis
  • Gate and barrier control systems
  • Parking lot management
  • Intersection monitoring and coordination
  • Per-lane vehicle counting for traffic analysis

Target Users

Traffic engineers, transportation planners, parking facility managers, and municipal traffic control personnel responsible for efficient traffic management.
