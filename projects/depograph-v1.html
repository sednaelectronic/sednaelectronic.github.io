DepoGraph V1.0

3D Scanning System

← Back to Projects

Product Overview

DepoGraph V1.0 is a portable precision 3D scanning system designed for creating detailed point cloud representations of indoor environments. The system employs a two-axis mechanical gimbal mechanism with a laser distance sensor to systematically scan and map spatial data. Engineered specifically for indoor precision use cases, this compact and portable device offers exceptional accuracy for controlled environment applications.

Technical Specifications

The DepoGraph V1.0 features a scanning capability of up to 100 meters with a field of view of 180° horizontal and 50° vertical coverage. Operating at 12V with 3.5A current consumption, the system supports battery connection for true portability. Data transmission is wireless-only, ensuring flexible deployment without cable constraints. The system is compatible with Sedna Studio v1.0 software for data processing and visualization. Note that auto-merge functionality is not supported in this version.

Environmental Considerations: The system is optimized for indoor environments and is not recommended for use in dusty or harsh environmental conditions where sensor performance may be compromised.

Purpose and Use Cases

The system is designed for inventory management, volume calculation, space planning, documentation, and quality control applications. Its portable design and indoor-optimized performance make it ideal for accurate 3D mapping of storage facilities, warehouses, and other controlled indoor spaces where precision and portability are essential.

Key Applications

  • 3D mapping of storage facilities and warehouses
  • Accurate volume measurements of storage areas
  • Creating 3D models for layout optimization
  • Generating point cloud records of spaces
  • Verifying spatial dimensions and layouts

Target Users

Warehouse managers, logistics professionals, facility planners, and quality control specialists who need accurate spatial data for operational planning and documentation.
