Product Overview

LADAR v1 is a single-unit two-axis scanning system designed for open-area people passage detection and access control. The system uses a LiDAR distance sensor mounted on a motorized two-axis gimbal to perform raster scanning of a detection area, analyzing distance measurements to determine presence and passage events.

Purpose and Use Cases

The system enables reliable people detection and passage monitoring for access control, security applications, and occupancy monitoring in open areas without physical barriers.

Key Applications

  • Entry and exit point monitoring
  • People movement detection in open areas
  • Access control system integration
  • Security and surveillance applications
  • Occupancy monitoring and counting

Target Users

Security professionals, facility managers, access control system integrators, and safety coordinators requiring reliable people detection capabilities.
