Ladar 2D Scanner

Perimeter Protection System

← Back to Projects

Product Overview

The Ladar 2D Scanner is a professional perimeter protection system designed for open environments, industrial sites, and warehouses. The system performs scanning and detection of unauthorized persons and objects using a non-visible wavelength laser, providing reliable security monitoring without visual intrusion. The system features a 360-degree Field of View (FOV) for full-area coverage, enabling comprehensive perimeter protection with a single unit. With a maximum operational range of up to 180 meters and rotational scanning speed of 2 Hz, the system delivers continuous monitoring and rapid detection response for security-critical applications.

Technical Specifications

The Ladar 2D Scanner operates at 12V with 3.5A current consumption, providing efficient power utilization for continuous operation. The system features dual communication interfaces: LAN connectivity for stable data communication and integration with security infrastructure, and RS485 interface for reliable industrial communication protocols. This dual-interface design ensures flexible integration with various security and monitoring systems.

Networking and Scalability: The system supports networking of multiple devices for extended coverage, enabling comprehensive perimeter protection across large industrial sites and warehouse facilities. Multiple units can be coordinated to provide seamless coverage of extended perimeters, with each unit contributing to a unified security network.

Software Integration: The Ladar 2D Scanner is fully compatible with Sedna Protecting Manager software, providing centralized monitoring, configuration, and alarm management capabilities. This integration enables comprehensive security system management, allowing operators to monitor multiple detection points, configure system parameters, and manage alarm events from a single centralized platform.

Purpose and Use Cases

The system enables reliable detection and monitoring of unauthorized access for perimeter protection, security applications, and infrastructure protection in open areas. Its wide-area coverage, reliability, and suitability for security-critical applications make it an essential component of modern perimeter protection systems for industrial facilities, warehouses, and critical infrastructure sites.

Key Applications

  • Perimeter protection in open environments
  • Industrial site security monitoring
  • Warehouse access control and monitoring
  • Unauthorized person and object detection
  • Security and surveillance applications
  • Critical infrastructure protection
  • Multi-device networked security systems

Target Users

Security professionals, facility managers, access control system integrators, and safety coordinators requiring reliable people detection capabilities.
